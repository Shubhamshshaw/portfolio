## 🚀 Getting Started

### Option 1 — Just Open It

```bash
# Clone the repo
git clone https://github.com/Shubhamshshaw/portfolio.git

# Open in your browser
open index.html
```

That's it. No build step. No `npm install`. It just works.

### Option 2 — Serve Locally (recommended for video/audio)

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve .

# Then visit
http://localhost:8080
```

> ⚠️ Browsers block autoplay and some media APIs when opening HTML files directly from disk (`file://`). A local server ensures full functionality including video backgrounds and audio.

## 🎨 Design System

The portfolio uses a custom CSS variable palette — dark, neon, cyberpunk:

```css
--void:     #02020D   /* Background black — the void */
--cyan:     #00F5FF   /* Primary accent — electric cyan */
--violet:   #9D5CFF   /* Secondary accent — deep violet */
--magenta:  #FF00FF   /* Tertiary accent — hot magenta */
--green:    #00FF88   /* Status / success — neon green */
--white:    #EEF8FF   /* Body text — cool white */
```

**Typography Trio:**
- `Orbitron` — Display / headings (the sci-fi hero font)
- `Space Grotesk` — Body text (readable, modern)
- `Space Mono` — Labels, tags, nav items (monospace precision)
